## 建立向量庫

首先我們建立一個 `Vector`的`class`

``` javascript=1
function Vector(x,y){
    this.x=x;
    this.y=y;
}
```
接著建立一個方法可以得到向量的長度(見2-5式)

``` javascript=4
    this.getLengthSquare=function(){
        return this.x*this.x+this.y*this.y;
    }
    this.getLength=function(){
        return sqrt(this.getLengthSquare());
    }
```
